# Corazón de Pelos

Bienvenido al repositorio oficial del sitio web **Corazón de Pelos**. Este proyecto es un catálogo digital diseñado para mostrar accesorios importados
seleccionados con amor para mascotas.

## 🚀 Sobre el proyecto
Este sitio web permite a los usuarios visualizar de manera dinámica y atractiva la variedad de productos que ofrece la marca,
facilitando la navegación a través de un catálogo con carrusel de imágenes y un diseño adaptativo (responsive).

## 🛠 Tecnologías utilizadas
El proyecto fue desarrollado utilizando las tecnologías base de la web:
- **HTML5**: Estructura semántica del sitio.
- **CSS3**: Diseño, estilos y adaptabilidad (media queries).
- **JavaScript**: Lógica interactiva para el carrusel de productos.

## 📱 Características
- Diseño "Mobile First" (adaptado para celulares).
- Carrusel de productos interactivo.
- Interfaz intuitiva y moderna.
